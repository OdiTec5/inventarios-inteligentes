# inventarios-inteligentes
Se pretende crear una aplicacion que se le puede dar un uso real en un entorno real, el tener elcontrol del inventario es importante para entender la disponibilidad y/ola falta de algun activo o producto, mediante esta aplicación podremos gestionar todo lo referente al inventario.
Se usaran herramientas de desarrollo.
Interfaz: HTML — CSS — JavaScript.
Backend:	Node.js — Express.
BBDD: MongoDB o PostgreSQL.
Control de versiones: Git + GitHub.
Desplegar: Vercel / Render / Netlify.
IA responsiva: Llamadas a APIs / Integración IA.
